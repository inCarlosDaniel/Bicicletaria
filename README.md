# Bicicletaria



Site desenvolvido em HTML, CSS e JavaScript para apresentar modelos de bicicletas e pecas, com foco em navegacao simples, visual moderno e destaque dos produtos.

## Objetivo do Projeto



O projeto Bicicletaria tem como objetivo apresentar bicicletas e pecas de forma organizada, com paginas separadas por categoria, melhorando a experiencia de quem quer visualizar produtos e precos com clareza.



A aplicacao foi criada para resolver pontos como:



- navegacao confusa entre categorias;

- visual simples demais para vitrine de produtos;

- falta de destaque para promocoes e lancamentos.



---



## Principais Funcionalidades



### Pagina Inicial



Exibe promocoes e lancamentos em destaque com imagens e precos.



### Navegacao por Categorias



Acesso rapido as paginas de Montain Bikes, Speed Bikes, Gravel e Pecas.



### Destaque de Menu Ativo



Identificacao automatica da pagina atual no menu.



### Animacoes Visuais



Animacoes de entrada, efeitos de hover e interacao com imagens.



### Layout Responsivo



Adaptacao para desktop e dispositivos moveis.

## Pre-requisitos para Executar o Projeto



- Navegador atualizado (Google Chrome, Microsoft Edge, Firefox)

## Requisitos do Sistema



- interface visual agradavel e intuitiva;

- navegacao entre paginas por menu principal;

- exibicao de produtos com imagem e preco;

- responsividade para telas menores.
